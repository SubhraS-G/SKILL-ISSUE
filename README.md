## 🎮 About

SKILL ISSUE is a troll platformer where nothing is what it seems.
Fake checkpoints. Disappearing platforms. A finish line that runs away from you.

Built entirely from scratch in Unity 6 using C# — no templates, no pre-made scripts.
Every mechanic, every troll, every insult written by hand.

**Built as Project 3 of my Game Development Portfolio — CSE Final Year.**

---

## ✨ Features

- 🏃 **Animated Player** — Idle, Run, Jump, Fall animations with smooth transitions
- 🟫 **Falling Platforms** — shake before dropping, respawn after 3 seconds
- ⚡ **Spike Traps** — instant death, maximum rage guaranteed 😂
- ✅ **Fake Checkpoint** — flashes green, says "Saved!"... then kills you 😈
- 🏁 **Fake Finish Line** — runs away 3 times before letting you win
- 💬 **Troll Message System** — random insults every time you die
- ❤️ **Death & Respawn** — instant respawn to keep the rage going
- 💀 **Death Counter** — tracks every single death in real time
- 📊 **Persistent Stats** — total deaths saved between sessions
- 🎮 **Main Menu** — shows your total shame on the title screen
- 🏆 **Level Complete Screen** — congratulates you... sarcastically
- 🪟 **Windows + WebGL** — download or play in browser

## 🧠 What I Built & Learned

- Unity 6 Tilemap system — painting, colliders, composite collider
- Animator Controller — state machine with transitions & parameters
- Coroutines — shake effects, timed events, fake mechanics
- Physics2D — ground detection, trigger zones, rigidbody control
- PlayerPrefs — persistent data between sessions
- Scene management — MainMenu → Game → LevelComplete flow
- Singleton pattern — DeathManager, TrollMessageSystem, LevelComplete
- UI system — Canvas scaling, TextMeshPro, buttons, panels
- Git version control for Unity projects
- Windows .exe & WebGL builds
